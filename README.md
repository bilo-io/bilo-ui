# XUI

This is an npm package used for frontend web development.

## Quickstart

To dev on the library in watch mode, run:

- `npm install`
- `npm run dev`

To run it locally, run:

- `npm link xui`
- `npm unlink`

> NOTE: In project directory running `npm unlink xui`

To unlink it globally, run:

- `npm remove -g xui`
